### install meteor 1.7.0.5
```
sh meteor-1.7.0.5-install.sh
```

### unzip meteor old version packages for 1.7.0.5
```
open ~/.meteor/packages/
double click meteor-packages.zip file to unzip all packages to ~/.meteor/packages/
```
